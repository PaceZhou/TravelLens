## 📱 Обзор проекта

TravelLens - это **приложение-путеводитель для фотографии в путешествиях с картой в приоритете**, которое помогает пользователям находить лучшие места для фотосъемки и местную кухню.

**Ключевые функции**:
- 🎁 **Загадочная коробка путешествий** - Рекомендации направлений на основе ИИ
- 📍 **Исследование карты** - Ближайшие достопримечательности с расстоянием в реальном времени
- 📷 **Гид по фототочкам** - Детальные советы по времени и композиции
- 🍜 **Рекомендации еды** - Местные рестораны рядом с достопримечательностями

---

## 🏗️ Технологический стек

**Frontend**: React 18 + TypeScript + Vite + Capacitor  
**Backend**: NestJS + PostgreSQL + PostGIS  
**Платформы**: Web + iOS + Android + HarmonyOS

## 🚀 Быстрый старт

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run start:dev
```
