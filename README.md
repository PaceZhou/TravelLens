## 📱 项目简介

TravelLens 是一款**地图优先的旅行出片指南应用**，帮助用户发现最佳拍照机位和周边美食。

**核心亮点**:
- 🎁 **旅行盲盒** - 智能算法推荐意想不到的目的地
- 📍 **地图探索** - 实时显示附近景点和距离
- 📷 **机位指南** - 详细的拍照时间、构图建议
- 🍜 **美食推荐** - 景点周边特色餐厅

---

## 🏗️ 技术栈

**前端**: React 18 + TypeScript + Vite + Capacitor  
**后端**: NestJS + PostgreSQL + PostGIS  
**平台**: Web + iOS + Android + 鸿蒙

---

## 🚀 快速开始

### 前端

```bash
cd frontend
npm install
npm run dev
```

### 后端

```bash
cd backend
npm install
npm run start:dev
```
