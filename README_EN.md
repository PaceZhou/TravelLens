## 📱 Overview

TravelLens is a **map-first travel photography guide app** that helps users discover the best photo spots and local cuisine.

**Key Features**:
- 🎁 **Travel Mystery Box** - AI-powered destination recommendations
- 📍 **Map Exploration** - Real-time nearby spots with distance
- 📷 **Photo Spot Guide** - Detailed timing and composition tips
- 🍜 **Food Recommendations** - Local restaurants near attractions

---

## 🏗️ Tech Stack

**Frontend**: React 18 + TypeScript + Vite + Capacitor  
**Backend**: NestJS + PostgreSQL + PostGIS  
**Platforms**: Web + iOS + Android + HarmonyOS

---

## 🚀 Quick Start

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run start:dev
```
