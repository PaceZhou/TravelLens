## 🔄 内容采集工作流程

### 1. 采集阶段 (采集员)

**工具**: Excel 模板 + 图片文件夹

**Excel 字段**:
```
景点ID | 名称 | 城市 | 地址 | 纬度 | 经度 | 门票 | 开放时间 | 
历史介绍 | 最佳季节 | 游览时长 | 标签 | 
机位1名称 | 机位1时间 | 机位1建议 | 
机位2名称 | 机位2时间 | 机位2建议 | 
美食1名称 | 美食1描述 | 美食1价格 |
美食2名称 | 美食2描述 | 美食2价格
```

**图片命名规范**:
```
景点ID_cover.jpg          # 封面
景点ID_photo1.jpg         # 机位1
景点ID_photo2.jpg         # 机位2
景点ID_photo3.jpg         # 机位3
景点ID_food1.jpg          # 美食1
景点ID_food2.jpg          # 美食2
```

---

### 2. 审核阶段 (文案编辑)

**检查清单**:
- [ ] 图片清晰度 (≥1920x1080)
- [ ] 文字无错别字
- [ ] GPS 坐标准确
- [ ] 信息完整度 ≥90%

**审核时间**: 每个景点 15 分钟

---

### 3. 美化阶段 (图片编辑)

**处理内容**:
- 色彩校正
- 裁剪构图
- 添加水印 (可选)
- 压缩优化 (Web: 200KB, 原图: 2MB)

**工具**: Photoshop / Lightroom

---

### 4. 导入阶段 (后端开发)

**导入脚本**:
```bash
npm run import:spots -- --file spots_week1.xlsx
```

**验证**:
- 数据库记录数
- 图片 CDN 上传成功
- 地图标记点显示

---

## 📈 每日工作节奏

### 采集员 (每人每天)

**上午 (9:00-12:00)**:
- 实地探访 2 个景点
- 拍摄 10-15 张照片
- 记录基础信息

**下午 (14:00-18:00)**:
- 整理照片
- 填写 Excel
- 提交审核

**产出**: 2-3 个景点/人/天

---

### 文案编辑 (每天)

**上午 (9:00-12:00)**:
- 审核前一天提交的景点
- 修改文案

**下午 (14:00-18:00)**:
- 撰写推荐理由模板
- 优化历史介绍

**产出**: 审核 5-8 个景点/天

---

### 图片编辑 (每天)

**全天 (9:00-18:00)**:
- 处理 30-40 张图片
- 批量压缩优化

**产出**: 5-8 个景点的图片/天
