# 🎁 旅行盲盒 - 核心亮点功能总结

## 一、功能定位

**名称**: "去哪儿盲盒" / Travel Mystery Box  
**Slogan**: "让算法带你去未知的远方"

**核心价值**:
- 🎲 打破选择困难症
- ✨ 发现意想不到的目的地
- 🧠 智能推荐 + 惊喜感
- 🎨 极致的视觉体验

---

## 二、算法核心

### 多维度评分系统

```
最终得分 = 
  个性化匹配 (40%) +
  时间适宜度 (25%) +
  距离合理性 (20%) +
  热度平衡 (10%) +
  随机惊喜 (5%)
```

**不是纯随机，而是科学推荐！**

---

## 三、技术实现

### 前端
- React + Framer Motion (3D 动画)
- 3 秒抽取动画
- 卡片翻转效果

### 后端
- NestJS 算法服务
- PostgreSQL + PostGIS (地理计算)
- 加权随机选择
